-------------------------
LAUNCHING QPME IN ECLIPSE
-------------------------

- use provided qpme.launch file (right click --> run as --> QPME)
- if it doesn't work double check the following:
  o open run confgurations --> qpme
  o Main tab --> Program to Run --> Run a product --> QPE_Base.QPE
  o Main tab --> JRE --> Runtime JRE --> jre6
  o PLug-ins --> Workspace --> QPE_Base and QPE_SimQPN selected, Click on "Add Required Plug-ins".
  o Common --> Save as --> Shared file --> \qpe-base
