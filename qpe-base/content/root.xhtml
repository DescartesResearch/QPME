System Requirements

QPME is made of two components: QPE (QPN Editor) and SimQPN (Simulator for QPNs). QPE runs on all platforms supported by Eclipse including Windows, Linux, Solaris, HP-UX, IBM AIX and Mac OS. The only thing required is a Java Runtime Environment (JRE) 6.0 or later. It is recommended to run QPE on Windows since this is the platform it has been tested on. SimQPN can be run either as Eclipse plugin in QPE or as a standalone Java application.
